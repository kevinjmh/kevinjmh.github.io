Python修改pip源 加快安装速度

Linux下打开.pip/pip.conf，加入以下内容，以后使用sudo pip install xxxx的时候非常快：

[global]
index-url = http://mirrors.opencas.cn/pypi/

Windows下修改%HOME%\pip\pip.ini， 没有就自己创建一个

[global]
timeout = 60
index-url = http://mirrors.opencas.cn/pypi/

[install]
timeout = 30
find-links =
    http://pypi.douban.com
    http://pypi.v2ex.com/simple/
